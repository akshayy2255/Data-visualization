Problem Statement

Organizations across different industries provide different parental leave policies for their employees.
However, when this information is stored in large datasets, it becomes difficult to understand and compare
the benefits provided by each company or industry. Simply viewing the data in tabular form does not clearly
show patterns, trends, or differences in maternity and paternity leave policies.
The purpose of this project is to use data visualization techniques to analyze parental leave data and present
it in a clear and meaningful way. By creating graphical representations such as bar charts, histograms, scatter
plots, and heatmaps, the project helps identify trends in maternity and paternity leave across different
industries. This makes it easier to understand which industries provide better parental benefits and how leave
policies vary among companies.

Dataset Description

The dataset used in this project contains information about parental leave benefits provided by different
companies across various industries. The data is stored in a CSV (Comma-Separated Values) file, which is a
common format used for storing tabular data. Each row in the dataset represents a specific company, and
each column represents a particular type of information related to parental leave policies.

The dataset includes the following columns:

 Company – The name of the organization providing the leave policy.
 Industry – The industry to which the company belongs (for example: Accounting, Advertising, Business, Consulting, Educational).
 Paid Maternity Leave – The number of weeks of paid leave provided to mothers after childbirth.
 Unpaid Maternity Leave – The number of weeks of unpaid maternity leave available.
 Paid Paternity Leave – The number of weeks of paid leave provided to fathers.
 Unpaid Paternity Leave – The number of weeks of unpaid leave available for fathers.

For better analysis and visualization, only a few industries such as Accounting, Advertising, Business,
Consulting, and Educational were selected from the dataset.
